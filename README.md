# AI Resume Screening System

## 📌 Project Overview

AI Resume Screening System is a Machine Learning based web application developed using Streamlit.  
The system analyzes resumes, predicts suitable job roles, calculates ATS scores, and stores candidate information for HR management.

---

## 🎯 Aim

To automate the resume screening process using Artificial Intelligence and Machine Learning techniques for efficient candidate evaluation.

---

## ⚙️ Technologies Used

- Python
- Streamlit
- Machine Learning
- Scikit-learn
- Pandas
- SQLite
- PDFMiner
- Python-Docx

---

## 🧠 Algorithm Used

### Logistic Regression

Logistic Regression is used to classify resumes into different job categories based on resume content.

---

## 🚀 Features

- Resume Upload (PDF & DOCX)
- AI-based Job Role Prediction
- Resume ATS Score Calculation
- Skills Extraction
- HR Dashboard
- Candidate Database Storage
- Resume Search & Delete
- Job Recommendation System

---

## 📊 Output

- Predicted Job Role
- ATS Resume Score
- Recommended Skills
- Candidate Details Storage
- HR Analytics Dashboard

---

## 🗄️ Database

SQLite database is used to store:

- Candidate Name
- Email
- Phone Number
- Predicted Role
- ATS Score
- Upload Date
